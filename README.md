# ar_shooting_game

## Game Design Document

### Core loops

- Detect planes
- Spawn markers
- [Gameplay loop](#gameplay-loop)

#### Gameplay loop

- Player
    - move
    - shoot
    - take damage
    - kill NPC
    - destroy NPC spawners
- NPC Spawners
    - spawn (at marker locations)
    - take damage
    - spawn NPCs
- NPC
    - spawn
    - move
    - take damage
